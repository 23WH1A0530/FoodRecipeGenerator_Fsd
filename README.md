# 🍽️ Food Recipe Generator (RecipeVault)

A full-stack web application that allows users to search, create, and manage recipes based on ingredients.

---

## 🚀 Features

- 🔐 User Authentication (Login/Register)
- 🍲 Add & Manage Recipes
- 🔍 Ingredient-based Search
- ⭐ Favorites System
- 📊 Dashboard & Analytics
- 🛒 Shopping List & Pantry

---

## 🛠️ Tech Stack

**Frontend:**
- React (Vite)
- CSS

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB Atlas
